# VeritasVet Website

Static website for VeritasVet — Science-Driven Animal Nutrition.

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Import the repo on [vercel.com/new](https://vercel.com/new)
3. Vercel auto-detects it as a static site — click **Deploy**

Or deploy directly via CLI:
```bash
npm i -g vercel
cd veritasvet-site
vercel
```

## Adding Images

Place your images in `assets/images/` following the paths listed in `IMAGE-CHECKLIST.md`.
